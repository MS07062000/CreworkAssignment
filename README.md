## Installation

1. Clone the repository: `git clone https://github.com/MS07062000/CreworkAssignment.git`
2. Install dependencies: `npm install`

## Usage

1. Run the project: `npm run dev`
2. Open the browser and navigate to `http://localhost:3000`
